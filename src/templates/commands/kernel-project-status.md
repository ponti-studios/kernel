argumentHint: optional project id

View the current status and metadata of a project.

Output includes:
- Project goal and status
- Target date (if set)
- Parent initiative (if linked)
- File path to the project directory
- Any linked milestones

Use this to quickly check:
- What the project is about
- When the project is due
- Where the project files are stored
- Whether it's active, done, or archived
