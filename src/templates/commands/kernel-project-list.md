View all projects in this codebase.

Output includes:
- Project ID and goal
- Current status (active, done, archived)
- Parent initiative (if linked)

Use this to:
- Get an overview of all projects
- Find a project by goal
- See which projects are currently active
- Understand the project-to-initiative mapping
