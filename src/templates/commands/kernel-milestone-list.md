View all milestones in this codebase.

Output includes:
- Milestone ID and goal
- Current status (active, done, archived)
- Parent project (if linked)

Use this to:
- Get an overview of all milestones
- Find a milestone by goal
- See which milestones are currently active
- Understand the milestone-to-project mapping
