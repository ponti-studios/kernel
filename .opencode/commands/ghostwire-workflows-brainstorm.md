# Ghostwire: Workflows Brainstorm

Generate ideas and explore solutions.

## Process

1. Define Problem
   - What challenge to solve
   - Constraints to consider
   - Success criteria

2. Generate Ideas
   - Go wide first
   - No bad ideas
   - Build on others

3. Evaluate Options
   - Pros and cons
   - Feasibility
   - Tradeoffs

4. Select Approach
   - Recommend solution
   - Explain reasoning
   - Note risks
