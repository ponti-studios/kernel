# Examples Index

This directory contains example jinn configurations.

## Examples

### 1. Minimal (`minimal/`)
- Single tool (OpenCode)
- Core profile
- Basic commands

### 2. Multi-Tool (`multi-tool/`)
- Multiple tools
- Extended profile
- All features

### 3. Commands Only (`commands-only/`)
- Commands only, no skills
- Lightweight setup

## Usage

```bash
# Copy config to your project
cp examples/minimal/.jinn/config.yaml .jinn/

# Or use CLI
jinn init --tools opencode
```
